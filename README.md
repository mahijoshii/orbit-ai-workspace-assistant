# Orbit — AI Workspace Assistant

Orbit is an AI-powered productivity assistant built on top of Google Workspace.

It integrates with Google Calendar and Gmail to help users plan their day, detect follow-ups, and turn natural-language tasks into scheduled calendar blocks.

## Problem

Managing email, calendar events, and tasks is fragmented. People waste time deciding what to do next, remembering who to follow up with, and manually moving tasks into their calendar.

Orbit solves this by acting as an intelligent planning layer over Google Workspace.

## Features

- Google Calendar integration
- Gmail follow-up detection
- AI-generated daily planning
- Natural-language task scheduling
- Calendar event creation
- Approval-first workflow before actions are taken

## Tech Stack
